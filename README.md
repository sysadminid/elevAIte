elevAIte with Dicoding - Source Code Repository
--

### Overview
This repository hosts the source code developed during the elevAIte with Dicoding AI training program. The program is an initiative by Microsoft to support the ASEAN Digital Masterplan 2025, aiming to equip digital talents in the ASEAN region, particularly in Indonesia, with advanced AI skills.


### Notes
- This repository contains the official source code from the training and does not accept external contributions.
- The code is provided for reference and educational purposes.
